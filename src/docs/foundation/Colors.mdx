# Colors

Our color system provides a comprehensive palette for building consistent, accessible interfaces.

## Color Philosophy

- **Systematic** - Predictable scales from light to dark
- **Accessible** - WCAG AA compliant contrast ratios
- **Semantic** - Meaningful names for common use cases
- **Flexible** - Easy to extend and customize

---

## Base Color Scales

### Gray Scale

The foundation of our color system. Used for text, borders, and backgrounds.

<ColorScale colorName="gray" shades={[50, 100, 200, 300, 400, 500, 600, 700, 800, 900, 950]} />

**Usage:**
- `gray.50-200` - Backgrounds and surfaces
- `gray.300-400` - Borders and dividers
- `gray.500-900` - Text and icons

---

### Primary (Blue)

Our brand color, used for primary actions and interactive elements.

<ColorScale colorName="blue" shades={[50, 100, 200, 300, 400, 500, 600, 700, 800, 900, 950]} />

**Usage:**
- `blue.50-200` - Subtle backgrounds
- `blue.600` - Primary buttons and links (default)
- `blue.700-800` - Hover and active states

## Usage in Code

### With Panda CSS

```typescript
import { css } from '../styled-system/css';

const styles = css({
  bg: 'primary.default',
  color: 'text.inverse',
  _hover: {
    bg: 'primary.hover',
  }
});
```

### Programmatic Access

```typescript
import { token } from '../styled-system/tokens';

const primaryColor = token('colors.blue.600');
```
