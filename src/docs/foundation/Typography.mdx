## Font Weights

Available font weights for emphasis and hierarchy:

- **thin** (100) - Rarely used
- **light** (300) - Subtle text
- **normal** (400) - Body text (default)
- **medium** (500) - Slightly emphasized
- **semibold** (600) - Headings and labels
- **bold** (700) - Strong emphasis
- **extrabold** (800) - Very strong emphasis
- **black** (900) - Maximum emphasis

---

## Line Heights

Control vertical rhythm and readability:

- **none** (1) - Tight, for headings
- **tight** (1.25) - Compact
- **snug** (1.375) - Slightly relaxed
- **normal** (1.5) - Body text (default)
- **relaxed** (1.625) - Comfortable reading
- **loose** (2) - Maximum spacing

---

## Usage Guidelines

### Headings

```typescript
// H1 - Page titles
fontSize: '3xl',
fontWeight: 'bold',
lineHeight: 'tight'

// H2 - Section titles
fontSize: '2xl',
fontWeight: 'semibold',
lineHeight: 'tight'

// H3 - Subsection titles
fontSize: 'xl',
fontWeight: 'semibold',
lineHeight: 'snug'
```

### Body Text

```typescript
// Regular body
fontSize: 'md',
fontWeight: 'normal',
lineHeight: 'normal'

// Small text
fontSize: 'sm',
fontWeight: 'normal',
lineHeight: 'normal'
```

### UI Elements

```typescript
// Buttons
fontSize: 'md',
fontWeight: 'semibold',
lineHeight: 'none'

// Labels
fontSize: 'sm',
fontWeight: 'medium',
lineHeight: 'tight'
```
