# Design System

A comprehensive design token system built with Panda CSS for consistent, scalable interfaces.

## What is a Design System?

A design system is a collection of reusable components, guided by clear standards, that can be assembled to build applications.

### Our Principles

- **Consistency** - Single source of truth for design decisions
- **Accessibility** - WCAG AA compliant by default
- **Scalability** - Easy to extend and customize
- **Type Safety** - Full TypeScript support with autocomplete
- **Performance** - Zero-runtime CSS generation

### [Spacing](/?path=/docs/foundation-spacing--docs)

### [Shadows](/?path=/docs/foundation-shadows--docs)

- Shadow scale (xs → 2xl)
- Usage guidelines

## Quick Reference

### Using Tokens

```typescript
import { css } from '../styled-system/css';

const styles = css({
  // Colors
  bg: 'primary.default',
  color: 'text.inverse',
  
  // Spacing
  p: '4',
  m: '2',
  
  // Typography
  fontSize: 'md',
  fontWeight: 'semibold',
  
  // Shadows
  boxShadow: 'md',
  
  // Border radius
  borderRadius: 'lg',
});
```

### Component Variants

```typescript
import { cva } from '../styled-system/css';

const button = cva({
  base: {
    px: '4',
    py: '2',
    borderRadius: 'md',
  },
  variants: {
    variant: {
      primary: { bg: 'primary.default' },
      secondary: { bg: 'bg.muted' },
    }
  }
});
```
